USAGE NOTES
───────────
1. Default mark: assets/mark.svg or mark_black.png
2. ≤16px contexts: switch to mark_compact_*.png (no faded dot)
3. App icon: use mark_black.png on the existing app's icon background
4. Wordmark: 'Hachibu' set in Lato Light (or any clean sans, light weight)

The faded 8th dot is the brand thesis — 80% on a 9-cell grid is impossible,
and that approximation IS the point.
