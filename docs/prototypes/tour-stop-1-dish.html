📝 デザインノート (料理タブ版)

  • 食材タブ版との違い: kcal scale が 0-1000 (料理は1食まるごとなので桁が大きい)
  • 長押しバケットが半数 (中華麺/寿司/ピザ/定食汁): 料理タブは種類で値が大きく変動するbucketが多い。点線バーで「長押しで選択」を明示
  • PFCタグ: 料理は米/麺/パン+主菜+油の組合せでC+F+Pになりがち。タグは「C多め/C主体/C+F/F+C/バランス」で差別化
  • 料理代表値の例:
    • 丼 default = gyudon_class (660kcal) — 牛丼/親子丼系
    • カレー default = curry_class (720kcal) — 普通カレーライス
    • 和麺 default = udon (510kcal) — うどん
    • パスタ default = pasta_tomato (680kcal) — トマト系
    • サンド default = cold_sand (345kcal) — コンビニサンド
